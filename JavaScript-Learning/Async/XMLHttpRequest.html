The open() method is used to initialize a request

The send() method is used to send the HTTP request to the server after it has been configured using the open() method.

The purpose of creating an XMLHttpRequest object is to enable communication between the browser and the server,


The onreadystatechange event is triggered whenever the readyState property of the XMLHttpRequest object changes. This event is used to monitor the state of the HTTP request and handle the response asynchronously.
