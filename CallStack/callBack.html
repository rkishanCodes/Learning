Callback function A callback function is a function passed into another function as an argument, which is then invoked inside the outer function to complete some kind of routine or action.

Types of Callbacks:
a. Synchronous Callbacks:
These callbacks are executed immediately by the function to which they are passed. They don't involve any asynchronous operations.

Asynchronous Callbacks:
These callbacks are executed at a later time, typically after some asynchronous operation completes, such as fetching data from a server or reading a file.

5. Nested Callbacks (Callback Hell):
When dealing with multiple asynchronous operations, nesting callbacks can lead to deeply nested code, also known as "callback hell." This makes code hard to read and maintain.
